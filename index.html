🎵 ChordWatcher

A music scanner that puts notes from documents into a MIDI file.

View on GitHub

About

ChordWatcher reads notated music from documents — sheet music, chord charts, or scanned pages — and converts the notes it finds into a playable MIDI file.

Features

  • Scans .txt, .docx, and .pdf documents, including scanned PDFs via OCR.
  • Detects chord charts (C  G  Am  F) as well as explicit note names (C4 E4 G4).
  • Exports a standard .mid file ready to import into GarageBand or any DAW.
  • Comes with both a simple desktop GUI and a command-line interface.

How it works

  1. Scan a document containing musical notation.
  2. ChordWatcher detects the notes and chords on the page.
  3. The detected notes are written out as a MIDI file you can play or import into any DAW.

Get started

git clone https://github.com/zachattack993/chordwatcher.git
cd chordwatcher
pip install -r requirements.txt

python3 main.py song.txt -o song.mid

Running python3 main.py with no arguments opens the desktop GUI instead. Import the resulting .mid file into GarageBand via File > Import, then drag it onto a software instrument track.
