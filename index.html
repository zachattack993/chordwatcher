🎵 ChordWatcher

A music scanner that puts notes from documents into a MIDI file.

View on GitHub

About

ChordWatcher reads notated music from documents — sheet music, chord charts, or scanned pages — and converts the notes it finds into a playable MIDI file.

How it works

  1. Scan a document containing musical notation.
  2. ChordWatcher detects the notes and chords on the page.
  3. The detected notes are written out as a MIDI file you can play or import into any DAW.

Status

ChordWatcher is under active development. Check the GitHub repository for the latest progress.
